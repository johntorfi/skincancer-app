Welcome to the Skin Cancer Lesion Classifier App!

This project aims to achieve two objectives:

  1. Building a Deep Learning Model:

    Our primary goal is to develop a robust deep learning model capable of accurately classifying skin cancer. Through extensive research and analysis, I have developed and trained a deep learning model, utilizing the VGG16 architecture, and achieving a test accuracy of 0.7713429855217174.

  2. Building a REST API using Flask:

    In addition to the deep learning model, I have created a user-friendly REST API using Flask. This API serves as an interface for our deep learning model, allowing users to upload images of suspected areas. Once uploaded, the application analyzes the images and provides information regarding the type of skin cancer, if present.

    We are dedicated to providing a reliable and accurate solution for skin cancer classification. Please feel free to explore the app and upload your images for analysis.
